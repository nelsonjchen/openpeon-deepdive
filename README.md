# Deep Dive Banter

OpenPeon / CESP v1.0 sound pack built from verified AI podcast-host parody clips.

## Pack Files

- `openpeon.json`: CESP manifest
- `sounds/`: supported audio files for OpenPeon
- `source_clips/`: ignored local raw source material, not part of the tracked pack

## Included Categories

- `session.start`
- `task.acknowledge`
- `task.progress`
- `task.complete`
- `task.error`
- `input.required`
- `resource.limit`
- `session.end`

## Notes

- Formats: `.mp3`
- Individual files are well under `1 MB`
- Total pack size is under `50 MB`
- Raw source `.m4a` files are intentionally excluded from Git and kept in `source_clips/`
